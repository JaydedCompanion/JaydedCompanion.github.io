Super Cloud

Finished

Super Cloud is a 2D arcade game in which you have to dodge mountains by jumping at the right moment. You will play as the Pegboard Nerds, a EDM duo whose latest album is a fundraiser for cancer research. Use power ups and multipliers to get the highest score, and unlock new characters.

Project S

WIP

Project S is a shooter based in space, with very nice graphics. Shoot all kinds of weird enemies and fight against other enemy space ships.

Project S is currently being developed by HalfCloud Interactive.
